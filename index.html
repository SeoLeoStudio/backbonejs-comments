Backbone Comments

What

Backbone Comments is a small project that shows the power and the simplicity of a backbone application by creating a comments system.

How

  1. Install git
  2. Install node.js
  3. Install mongoDB

Now you have everything you need to start using and modifying bakconejs-comments:

// Clone the repo
$ git clone https://github.com/ricca509/backbonejs-comments.git
$ cd backbonejs-comments
// Install dependencies
$ npm install
// Launch the server
$ node server.js
